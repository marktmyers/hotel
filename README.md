# Mock Hotel Database

## Description
This is a final project for a Databases course at James Madison University. It is a mock database for a group of hotels with numerous tables for different aspects. Fake data is ready to be loaded in to use in the webapp.

## Languages and Utilities Used
- **HTML**
- **JavaScript**
- **SQL**
- **PostgreSQL**
- **Python**

## Environments Used
- **macOS**

## How to Use
1. Clone/download the repository to your local machine.
2. Load the data in load.sql based on the paths for your machines.
3. cd to /webapp.
4. Run python app.py.
5. Use any of the utilities to make queries through the app.
